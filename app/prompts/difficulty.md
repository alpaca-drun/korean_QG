# 난이도 등급 정의

| 등급 | 목표 정답률 | 핵심 특징 |
|------|-------------|----------|
| **최상** | 30% 미만 | 고차원적 사고(비판적·창의적), 복합 추론, 다단계 판단 |
| **상** | 30~50% | 추론적 이해, 정보 종합, 외적 준거 활용 |
| **중** | 50~70% | 사실적 이해, 기본 적용, 단일 정보 판단 |
| **하** | 70~85% | 명시적 정보 확인, 단순 이해 |
| **최하** | 85% 이상 | 지문에서 직접 확인 가능한 단순 문항 |

# 난이도 조절 핵심 전략

## 1. 정보 처리 복잡도 조절

| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| 정답 근거를 지문 한 곳에 집중 배치 | 정답 근거를 지문 여러 곳에 분산 배치 |
| 단일 정보로 정오 판단 가능 | 복수 정보 종합해야 정오 판단 가능 |
| 지문 표현을 선택지에 유사하게 사용 | 지문 표현을 선택지에서 다르게 변환 |

## 2. 선택지 구조 조절

| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| 단순 진술형: "~이다", "~했다" | 복합 진술형: "~하여(~를 통해) ~하고 있다" |
| 오답이 명백히 틀린 내용 | 오답의 앞부분은 맞고 뒷부분만 틀림 |
| 선택지 간 내용이 명확히 구분됨 | 선택지 간 내용이 유사하여 세밀한 판단 필요 |

## 3. <보기> 활용 조절

| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| <보기> 없이 지문만으로 해결 | <보기>를 외적 준거로 활용 |
| <보기>가 단순 사례 제시 | <보기>가 감상 관점/이론 제시 |
| <보기> 내용이 짧고 명확 | <보기> 내용이 길고 복합적 |

## 4. 인지적 요구 수준 조절

| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| 사실적 이해 (내용 확인) | 추론적 이해 (숨겨진 의미 파악) |
| 단순 적용 | 비판적·창의적 이해 |
| 명시적 정보 활용 | 함축적 정보 해석 |

# 영역별 난이도 조절 기법

## 1. 문학 영역

### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "윗글에 대한 설명으로 적절한 것은?"
- 선택지: 작품의 표면적 내용(화자, 시어, 사건) 직접 확인
- 구조: 단일 작품, <보기> 미사용
- 예시: "㉠에 담긴 화자의 정서로 적절한 것은?"

### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "<보기>를 참고하여 윗글을 감상한 내용으로 적절하지 않은 것은?"
- 선택지: 내용+형식 복합 진술 ("~를 통해 ~를 드러내고 있다")
- 구조: 복수 작품 비교, <보기>에 외적 준거(문학 이론, 시대 배경) 제시
- 예시: "<보기>의 관점에서 (가)와 (나)를 비교 감상한 내용으로..."

### 선택지 난이도 조절 예시

**낮은 난이도:**
① 화자는 자연을 예찬하고 있다.
② <제2장>에서는 자연의 이치를 제시하고 있다.

**높은 난이도:**
① <제2장>에서는 유사한 자연의 이치가 내포된 두 사례를 나란히 배열하여 화자의 기원을 드러내고 있다.
② 역설적 표현을 활용하여 화자의 내면적 갈등을 심화하고 있다.

## 2. 읽기 영역

#### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "윗글의 내용과 일치하는 것은?" / "윗글에 대한 설명으로 적절한 것은?"
- 선택지: 지문 정보의 단순 확인, 패러프레이징
- 구조: 단일 관점, 구체적 사례 중심 지문


#### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "윗글을 바탕으로 <보기>를 이해한 내용으로 적절하지 않은 것은?"
- 선택지: 지문 개념을 새로운 상황에 적용, 복수 정보 종합 판단
- 구조: 복수 관점 비교, 추상적 개념 + 구체적 사례의 입체적 구성
- <보기>: 지문 개념을 적용할 새로운 사례/상황 제시

### 선택지 난이도 조절 예시

**낮은 난이도:**
① 공공재는 비경합성을 갖고 있다.
② A 이론에서는 시장의 자율성을 강조한다.

**높은 난이도:**
① A사의 제조 공장에서 개개인의 성과로 계산하기 위해 산출되는 정보는, 윗글의 '중앙의 탑'에 있는 감시자가 '독방'을 향해 던지는 시선과 역할이 유사하다는 것을 전제하고 있다.
② 윗글의 관점에서 볼 때, <보기>의 사례는 정보 비대칭으로 인한 시장 실패의 전형적인 예에 해당한다.

## 3. 문법 영역

### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "<보기>의 ⓐ~ⓔ를 이해한 내용으로 적절한 것은?"
- 선택지: 단일 문법 개념 적용, 익숙한 예시 활용
- 구조: 하나의 문법 규칙만 적용
- 예시: "'읽히다'는 '읽다'의 어근에 접미사 '-히-'가 붙어 형성된 단어이다."

### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "<보기>를 참고할 때, 음운 변동 사례에 대해 바르게 이해한 것은?"
- 선택지: 복수 문법 개념 종합 적용 (예: 음운 변동 유형 + 음운 개수 변화)
- 구조: 여러 단계의 판단 과정 필요, 예외적 사례 포함
- 예시: 선택지에 추가 사례 제시하여 두 가지 내용을 동시에 판단하게 함

### 선택지 난이도 조절 예시

**낮은 난이도:**
① '읽히다'는 '읽다'의 어근에 접미사 '-히-'가 붙어 형성된 단어이다.
② ㉠에서는 음절 끝 자음이 대표음으로 바뀌는 음운 변동이 일어났다.

**높은 난이도:**
① ㉠: '값+도→[갑또]'에서처럼 음절 끝에 둘 이상의 자음이 오지 못하기 때문에 일어난 음운 변동이 있다.
② '흙하고[흐카고]'는 탈락 및 축약이 일어나 음운의 개수가 두 개 줄었군.

### 문법 문항 설계 시 유의점
- 개념어를 그대로 노출하기보다 풀어서 설명하는 것이 바람직함
- 중세 국어 자료 제시 시 현대어 풀이 병기 필요
- 선택지에 추가 사례를 포함하면 난도 상승

## 4. 매체 영역

#### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "위 자료에 대한 설명으로 적절한 것은?"
- 선택지: 매체 자료의 표면적 내용 확인
- 구조: 단일 매체 자료, 명시적 정보 중심
- 예시: "이 카드뉴스에서 사용한 시각 자료의 종류는?"

### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "(가)와 (나)를 비교하여 매체 특성에 따른 정보 전달 방식의 차이를 분석한 것으로 적절하지 않은 것은?"
- 선택지: 복수 매체 비교, 제작 의도와 표현 효과 연결
- 구조: 다양한 매체 자료 비교, 매체 특성과 내용의 관계 분석
- 예시: "블로그 글과 뉴스 기사의 정보 구성 방식을 비교한 내용으로..."

### 선택지 난이도 조절 예시

**낮은 난이도:**
① 이 영상에는 자막이 사용되었다.
② 카드뉴스는 이미지와 텍스트를 함께 사용하고 있다.

**높은 난이도:**
① (가)는 하이퍼링크를 활용하여 관련 정보에 대한 접근성을 높이고 있고, (나)는 댓글 기능을 통해 쌍방향 소통이 가능하도록 하고 있다.
② 정보 생산자의 의도가 매체 특성에 따라 다르게 구현되고 있음을 고려할 때, (가)의 편집 방식은...

## 5. 듣기·말하기 영역

### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "위 대화에서 확인할 수 있는 말하기 방식으로 적절한 것은?"
- 선택지: 발화 내용의 직접 확인, 단순 전략 파악
- 구조: 단순한 담화 상황 (일상 대화, 단순 발표)
- 예시: "발표자가 사용한 시각 자료의 종류로 적절한 것은?"


### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "[가]에 대한 설명으로 적절하지 않은 것은?" (청자 특성 고려)
- 선택지: 복합적 담화 전략 분석, 의도와 표현의 관계 파악
- 구조: 복잡한 담화 상황 (토론, 협상), 청자 반응 포함
- 예시: "~하기 위해[의도] ~하고 있다[표현]" 형식의 복합 진술

### 선택지 난이도 조절 예시

**낮은 난이도:**
① 청중에게 질문을 하고 있다.
② 시각 자료를 활용하고 있다.
③ 전문가의 견해를 인용하고 있다.

**높은 난이도:**
① 발표의 목적을 설명하기 위해 청중에게 특정 행동을 요구하고 있다.
② 발표 내용에 대한 신뢰도를 높이기 위해 전문가의 설명을 직접 인용하고 있다.
③ 발표 내용에 대한 청중의 이해 여부를 확인하기 위해 청중에게 질문을 하고 있다.

### 듣기·말하기 문항 설계 시 유의점

- 담화의 본질인 화자-청자 상호작용 반영 필요
- 청자의 반응(질문, 메모 등)을 포함하면 화법 특성 부각
- 선택지를 '의도+표현' 형태로 구성하면 난도 상승
- 독서 문항처럼 정보 나열식으로 설계하지 않도록 주의
  
## 6. 쓰기 영역

### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "(가)의 글쓰기 계획 중 (나)에 반영되지 않은 것은?"
- 선택지: 계획과 초고의 단순 대응 확인
- 구조: 계획→초고의 직접적 반영 여부 확인
- 예시: "고쳐쓰기 방안으로 적절한 것은?" (어휘/문장 수준)

### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "<보기>를 활용하여 <조건>에 따라 ㉠을 수정·보완한 것으로 가장 적절한 것은?"
- 선택지: 자료 해석 + 글 수정·보완 복합 판단
- 구조: 계획의 수정·조정 과정 반영, 복수 조건 충족 여부 판단
- 예시: "...하여 ~한다" 형식 (계획과 초고의 관련성 제시)

### 선택지 난이도 조절 예시

**낮은 난이도:**
① 해외에 있는 우리 문화재가 국내로 반환된 최근의 사례
② 맞춤법이 틀린 부분을 수정한다.
③ 문단의 통일성을 위해 벗어난 문장을 삭제한다.

**높은 난이도:**

① 덕종어보 환수와 관련된 내용을 언급하여 '서론'을 구체화한다.
② 따라서 종자 개발에 경쟁력 있는 종자 기업을 정책적으로 육성하되 연구 개발에 대한 투자와 지원을 강화해야 한다. 다만 개발된 종자의 장점만을 중시하여 종자를 획일화하는 것은 위험하므로 장기적 안목에서 다양한 종자의 개발과 확보에 노력해야 한다.

### 쓰기 문항 설계 시 유의점

- 글의 생산 과정을 중심으로 설계
- 계획→초고→수정의 역동적 과정 반영
- 고쳐쓰기는 어휘/문장/문단/글 수준을 고르게 포함
- 단순 정보 조회가 아닌 과정 간 관련성을 묻도록 설계

## 오답지 매력도 조절 기법

### 매력도 높은 오답지 만들기 (난이도 상승)

1. **부분 정답형**: 앞부분은 맞지만 뒷부분이 틀림
   ```
   "역설적 표현을 사용하여 [O] 지나온 삶에 대해 자책하는 화자의 모습을 드러내고 있다 [X]"
   ```

2. **정보 분산형**: 정오 판단을 위해 지문의 앞부분과 뒷부분을 모두 확인해야 함

3. **개념 혼동형**: 유사한 개념을 혼동하게 만듦
   ```
   "암묵지"와 "명시지"의 속성을 바꾸어 진술
   ```

4. **과잉 일반화형**: 부분적으로 맞는 내용을 전체에 적용
   ```
   "~도" → "~만" / "일부" → "모든"
   ```

### 매력도 낮은 오답지 만들기 (난이도 하락)

1. **명백한 오류형**: 지문 내용과 명확히 반대되는 진술
2. **무관 정보형**: 지문에서 전혀 다루지 않은 내용
3. **극단적 표현형**: "절대로", "반드시", "전혀" 등 극단적 표현 사용

## 발문 유형과 난이도

### 낮은 난이도 발문
- "윗글의 내용과 일치하는 것은?"
- "윗글에 대한 설명으로 적절한 것은?"
- "ⓐ~ⓔ에 대한 설명으로 적절한 것은?"
- "위 담화에서 확인할 수 있는 것은?"

### 중간 난이도 발문
- "윗글에 대한 이해로 적절하지 않은 것은?"
- "윗글의 서술상 특징으로 가장 적절한 것은?"
- "(가)와 (나)의 공통점으로 가장 적절한 것은?"
- "[가]에 대한 설명으로 적절하지 않은 것은?"

### 높은 난이도 발문
- "<보기>를 참고하여 윗글을 감상한 내용으로 적절하지 않은 것은?"
- "윗글을 바탕으로 <보기>를 이해한 내용으로 적절하지 않은 것은?"
- "<보기>의 관점에서 (가)와 (나)를 비교한 내용으로 적절하지 않은 것은?"
- "<보기>를 활용하여 <조건>에 따라 ㉠을 수정·보완한 것으로 가장 적절한 것은?"

## 문항 생성 체크리스트

### 난이도 확인 사항

- [ ] 요청된 난이도에 맞는 인지적 요구 수준인가?
- [ ] 정보 처리 복잡도가 난이도에 적합한가?
- [ ] 선택지 구조가 난이도에 맞게 설계되었는가?
- [ ] 오답지 매력도가 난이도에 적합한가?
- [ ] <보기> 활용 방식이 난이도에 부합하는가?

### 문항 품질 확인 사항

- [ ] 정답이 명확하고 오류가 없는가?
- [ ] 발문이 평가 요소를 명확히 드러내는가?
- [ ] 선택지 간 내용 중복이나 간섭이 없는가?
- [ ] 성취기준에 부합하는 문항인가?
- [ ] 지문 외 배경지식에 과도하게 의존하지 않는가?
- [ ] 영역 특성(화법의 상호작용, 작문의 과정 등)이 반영되었는가?

---

### 출력 예시 (난이도 '상' 문항)
```
<보기>를 참고하여 윗글을 감상한 내용으로 적절하지 않은 것은?

<보기>
[작품의 시대적 배경이나 작가의 의도를 설명하는 외적 준거 제시]

① [지문의 특정 표현] + [<보기> 관점에서의 해석] + [감상 진술]
② [복수 정보 종합이 필요한 감상 진술]
③ [부분 정답형 오답 - 앞부분 O, 뒷부분 X]
④ [정답 - <보기> 관점과 불일치하는 감상]
⑤ [지문 앞뒤 정보 조합 필요한 감상 진술]
```

## 영역별 핵심 요약

| 영역 | 난이도 상승 핵심 전략 |
|------|---------------------|
| **문학** | 외적 준거(<보기>) 활용, 복수 작품 비교, 내용+형식 복합 진술 |
| **읽기** | 개념의 새로운 상황 적용, 복수 관점 비교, 추상적 개념 추론 |
| **문법** | 복수 문법 개념 종합, 추가 사례 판단, 다단계 적용 |
| **매체** | 복수 매체 비교, 제작 의도와 표현 효과 연결, 매체 특성 분석 |
| **듣기·말하기** | 의도+표현 복합 진술, 청자 반응 포함, 화자-청자 상호작용 |
| **쓰기** | 자료 해석+글 수정 복합, 계획-초고 관련성, 복수 조건 충족 |
