# 난이도 등급 정의

| 등급 | 목표 정답률 | 핵심 특징 |
|------|-------------|----------|
| **최상** | 30% 미만 | 고차원적 사고(비판적·창의적), 복합 추론, 다단계 판단 |
| **상** | 30~50% | 추론적 이해, 정보 종합, 외적 준거 활용 |
| **중** | 50~70% | 사실적 이해, 기본 적용, 단일 정보 판단 |
| **하** | 70~85% | 명시적 정보 확인, 단순 이해 |
| **최하** | 85% 이상 | 지문에서 직접 확인 가능한 단순 문항 |

# 난이도 조절 핵심 전략

## 1. 정보 처리 복잡도 조절

| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| 정답 근거를 지문 한 곳에 집중 배치 | 정답 근거를 지문 여러 곳에 분산 배치(예: 1문단 정보 + 4문단 정보를 결합해야 판단 가능) |
| 단일 정보로 정오 판단 가능 | 복수 정보 종합해야 정오 판단 가능. 선택지 내 "~하고, ~하다" 구조에서 두 정보의 근거가 지문에서 물리적으로 멀리 떨어질수록, 또는 논리적 연결이 느슨할수록 난이도 상승 |
| 지문 표현을 선택지에 유사하게 사용 | 지문 표현을 선택지에서 다르게 변환(예: 지문의 "A는 B를 억제한다" → 선택지의 "B의 활성화는 A로 인해 제한된다"처럼 구조·어휘를 바꾸어 표현) |

## 2. 선택지 구조 조절

| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| 단순 진술형: "~이다", "~했다" | 복합 진술형: "~하여(~를 통해) ~하고 있다" |
| 오답이 명백히 틀린 내용 | 오답의 앞부분은 맞고 뒷부분만 틀림 |
| 선택지 간 내용이 명확히 구분됨 | 선택지 간 내용이 유사하여 세밀한 판단 필요 |

## 3. <보기> 활용 조절

| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| <보기> 없이 지문만으로 해결 | <보기>를 외적 준거로 활용 |
| <보기>가 단순 사례 제시 | <보기>가 감상 관점/이론 제시 |
| <보기> 내용이 짧고 명확 | <보기> 내용이 길고 복합적 |

## 4. 인지적 요구 수준 조절

| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| 사실적 이해 (내용 확인) | 추론적 이해 (숨겨진 의미 파악) |
| 단순 적용 | 비판적·창의적 이해 |
| 명시적 정보 활용 | 함축적 정보 해석 |

문학·문법 영역의 인지적 요구 수준은 기억→이해→적용→분석→평가→창안 순으로 높아지며, 난이도 '상' 이상에서는 분석·평가 수준의 문항을 포함하도록 한다. 듣기·말하기·매체 영역은 분석·평가 수준을 난이도 상승의 핵심 기준으로 삼는다. 쓰기 영역은 5단계 writing_phase 구조를 인지적 심화 기준으로 대신한다. 읽기 영역은 아래 읽기 영역 난이도 상승 요인을 우선 적용한다

## 5. 배경지식과 언어자료 결합 조절
| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|
| 지문 내 정보만으로 정오 판단 가능 | <보기>에 외적 준거(학습요소 범위 내 이론·개념)를 명시적으로 제시하고, 이를 지문 내용과 연결하여 판단 |
| 배경지식 없이도 풀 수 있는 구조 | <보기> 없이는 풀 수 없도록 설계하여 순수 배경지식 의존 방지 |

# 영역별 난이도 조절 기법

## 1. 문학 영역

### 낮은 난이도 (하~중) 문항 생성 시
- 발문: "윗글에 대한 설명으로 적절한 것은?"
- 선택지: 작품의 표면적 내용(화자, 시어, 사건) 직접 확인
- 구조: 단일 작품, <보기> 미사용
- 예시: "㉠에 담긴 화자의 정서로 적절한 것은?"

### 높은 난이도 (상~최상) 문항 생성 시
- 발문: "<보기>를 참고하여 윗글을 감상한 내용으로 적절하지 않은 것은?"
- 선택지: 내용+형식 복합 진술 ("~를 통해 ~를 드러내고 있다")
- 구조: 복수 작품 비교, <보기>에 외적 준거(문학 이론, 시대 배경) 제시
- 예시: "<보기>의 관점에서 (가)와 (나)를 비교 감상한 내용으로..."

### 선택지 난이도 조절 예시
**낮은 난이도:**
① 화자는 자연을 예찬하고 있다.
② <제2장>에서는 자연의 이치를 제시하고 있다.

**높은 난이도:**
① <제2장>에서는 유사한 자연의 이치가 내포된 두 사례를 나란히 배열하여 화자의 기원을 드러내고 있다.
② 역설적 표현을 활용하여 화자의 내면적 갈등을 심화하고 있다.

## 2. 읽기 영역

#### 낮은 난이도 (하~중) 문항 생성 시
- 발문: "윗글의 내용과 일치하는 것은?" / "윗글에 대한 설명으로 적절한 것은?"
- 선택지: 지문 정보의 단순 확인, 패러프레이징
- 구조: 단일 관점, 구체적 사례 중심 지문

#### 중간 난이도 (중) 문항 생성 시
- 발문: "윗글에 대한 이해로 적절하지 않은 것은?" /  "윗글에서 추론한 내용으로 적절하지 않은 것은?"
- 선택지: 지문의 두 곳 이상 정보를 연결해야 판단 가능, 지문 표현을 선택지에서 부분적으로 변환
- 구조: 단일 관점이지만 추론 필요, 명시적 정보와 비명시적 정보 혼합

#### 높은 난이도 (상~최상) 문항 생성 시
- 발문: "윗글을 바탕으로 <보기>를 이해한 내용으로 적절하지 않은 것은?"
- 선택지: 지문 개념을 새로운 상황에 적용, 복수 정보 종합 판단
- 구조: 복수 관점 비교, 추상적 개념 + 구체적 사례의 입체적 구성
- <보기>: 지문 개념을 적용할 새로운 사례/상황 제시
  
### 읽기 영역 난이도 상승 요인
읽기 영역의 난이도는 아래 요인의 복합 정도에 따라 결정된다. 해당 난이도에 맞는 요인을 선택하여 문항을 설계하라
| 난이도 | 주요 상승 요인 |
|----------------|------------------|
| 중 | 중심 내용과 세부 사례의 구분이 필요할 때 / 문단 간 관계를 파악해야 할 때 / 지문 표현을 선택지에서 부분적으로 변환할 때 |
| 상 | 글쓴이의 입장과 제시된 타인의 입장을 구별해야 할 때 / 조건부 진술의 의미를 해석해야 할 때 / 명시된 정보와 명시되지 않은 정보를 결합해야 할 때 |
| 최상 | 반론-재반론 구조에서 필자의 최종 입장을 파악해야 할 때 / 복수 관점을 비교·대조해야 할 때 / <보기>의 새로운 사례에 지문 개념을 적용해야 할 때 / 위 요인이 2개 이상 복합될 때 |

### 선택지 난이도 조절 예시

**낮은 난이도:**
① [개념어]는 [단일 속성]을 갖는다. →  지문 한 곳에서 확인 가능
② [A]는 [B]의 특징을 지닌다. → 지문 표현과 유사하게 바꾸어 진술

**중간 난이도:**
① [A]는 [B]와 달리 [속성C]를 가지며, [조건D]에서만 작동한다. 
→ 두 곳 이상의 정보를 연결해야 판단 가능

**높은 난이도:**
① [지문 개념]을 [<보기> 상황]에 적용할 때, [조건A]이면서 동시에 [조건B]인 것은 [결론C]에 해당한다. 
→  복수 정보 종합 + 새로운 상황 적용 필요

## 3. 문법 영역

### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "<보기>의 ⓐ~ⓔ를 이해한 내용으로 적절한 것은?"
- 선택지: 단일 문법 개념 적용, 익숙한 예시 활용
- 구조: 하나의 문법 규칙만 적용
- 예시: "'읽히다'는 '읽다'의 어근에 접미사 '-히-'가 붙어 형성된 단어이다."

### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "<보기>를 참고할 때, 음운 변동 사례에 대해 바르게 이해한 것은?"
- 선택지: 복수 문법 개념 종합 적용 (예: 음운 변동 유형 + 음운 개수 변화)
- 구조: 여러 단계의 판단 과정 필요, 예외적 사례 포함
- 예시: 선택지에 추가 사례 제시하여 두 가지 내용을 동시에 판단하게 함

### 선택지 난이도 조절 예시

**낮은 난이도:**
① '읽히다'는 '읽다'의 어근에 접미사 '-히-'가 붙어 형성된 단어이다.
② ㉠에서는 음절 끝 자음이 대표음으로 바뀌는 음운 변동이 일어났다.

**높은 난이도:**
① ㉠: '값+도→[갑또]'에서처럼 음절 끝에 둘 이상의 자음이 오지 못하기 때문에 일어난 음운 변동이 있다.
② '흙하고[흐카고]'는 탈락 및 축약이 일어나 음운의 개수가 두 개 줄었군.

### 문법 문항 설계 시 유의점
- 개념어를 그대로 노출하기보다 풀어서 설명하는 것이 바람직함
- 중세 국어 자료 제시 시 현대어 풀이 병기 필요
- 선택지에 추가 사례를 포함하면 난도 상승

## 4. 매체 영역

#### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "위 자료에 대한 설명으로 적절한 것은?"
- 선택지: 매체 자료의 표면적 내용 확인
- 구조: 단일 매체 자료, 명시적 정보 중심
- 예시: "이 카드뉴스에서 사용한 시각 자료의 종류는?"

### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "(가)와 (나)를 비교하여 매체 특성에 따른 정보 전달 방식의 차이를 분석한 것으로 적절하지 않은 것은?"
- 선택지: 복수 매체 비교, 제작 의도와 표현 효과 연결
- 구조: 다양한 매체 자료 비교, 매체 특성과 내용의 관계 분석
- 예시: "블로그 글과 뉴스 기사의 정보 구성 방식을 비교한 내용으로..."

### 선택지 난이도 조절 예시

**낮은 난이도:**
① 이 영상에는 자막이 사용되었다.
② 카드뉴스는 이미지와 텍스트를 함께 사용하고 있다.

**높은 난이도:**
① (가)는 하이퍼링크를 활용하여 관련 정보에 대한 접근성을 높이고 있고, (나)는 댓글 기능을 통해 쌍방향 소통이 가능하도록 하고 있다.
② 정보 생산자의 의도가 매체 특성에 따라 다르게 구현되고 있음을 고려할 때, (가)의 편집 방식은...

## 5. 듣기·말하기 영역

### 낮은 난이도 (하~중) 문항 생성 시

- 발문: "위 대화에서 확인할 수 있는 말하기 방식으로 적절한 것은?"
- 선택지: 발화 내용의 직접 확인, 단순 전략 파악
- 구조: 단순한 담화 상황 (일상 대화, 단순 발표)
- 예시: "발표자가 사용한 시각 자료의 종류로 적절한 것은?"


### 높은 난이도 (상~최상) 문항 생성 시

- 발문: "[가]에 대한 설명으로 적절하지 않은 것은?" (청자 특성 고려)
- 선택지: 복합적 담화 전략 분석, 의도와 표현의 관계 파악
- 구조: 복잡한 담화 상황 (토론, 협상), 청자 반응 포함
- 예시: "~하기 위해[의도] ~하고 있다[표현]" 형식의 복합 진술

### 선택지 난이도 조절 예시

**낮은 난이도:**
① 청중에게 질문을 하고 있다.
② 시각 자료를 활용하고 있다.
③ 전문가의 견해를 인용하고 있다.

**높은 난이도:**
① 발표의 목적을 설명하기 위해 청중에게 특정 행동을 요구하고 있다.
② 발표 내용에 대한 신뢰도를 높이기 위해 전문가의 설명을 직접 인용하고 있다.
③ 발표 내용에 대한 청중의 이해 여부를 확인하기 위해 청중에게 질문을 하고 있다.

### 듣기·말하기 문항 설계 시 유의점

- 담화의 본질인 화자-청자 상호작용 반영 필요
- 청자의 반응(질문, 메모 등)을 포함하면 화법 특성 부각
- 선택지를 '의도+표현' 형태로 구성하면 난도 상승
- 독서 문항처럼 정보 나열식으로 설계하지 않도록 주의
- 담화와 관련된 새로운 담화나 정보를 <보기>로 제시하면 난이도가 상승함.
- 담화를 구조와 관련지어 파악하게 하면 난이도가 상승함.
  
## 6. 쓰기 영역

### 쓰기 영역 난이도 조절 시 추가 고려사항 
쓰기 영역은 5단계 학습 활동 과정(계획하기 → 내용 생성하기 → 내용 조직하기 → 표현하기 → 고쳐쓰기)으로 구성되며, 각 단계별로 난이도 조절 방식이 다르다. 

#### 단계별 난이도 상승 핵심 요인
| 쓰기 단계 | 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) |
|----------------|------------------|------------------|
| 계획하기 | 단일 요소 판단 | 복수 요소 종합 판단 (목적+독자+매체 동시 고려) |
| 내용 생성하기 | 단일 자료 활용 | 복수 자료 종합 활용 |
| 내용 조직하기 | 표면적 순서 판단 | 위계 구조 및 논리적 연결 판단 |
| 표현하기  | 명시적 오류 수정 | 문맥 고려 정교한 표현 선택 |
| 고쳐쓰기 | 단일 조건 충족 | 복수 조건 동시 충족 + 수정 방향 타당성 판단|

### 낮은 난이도 (하~중) 문항 생성 시
- 발문: "(가)의 글쓰기 계획 중 (나)에 반영되지 않은 것은?", "다음 자료의 내용으로 적절한 것은?", "ⓐ~ⓔ를 고쳐 쓰기 위한 방안으로 적절한 것은?"
- 선택지: 계획과 초고의 단순 대응 확인, 단일 자료의 직접 확인
- 구조: 계획→초고의 직접적 반영 여부 확인, 단일 단계만 측정(예: 계획하기 단계에서 ‘목적’만 판단), 고쳐쓰기는 어휘/문장 수준의 단순 수정
- 예시: 글쓰기 계획 에서 ‘예상 독자’로 설정한 대상은?, 맞춤법이 틀린 부분을 수정한 것은?
  
### 높은 난이도 (상~최상) 문항 생성 시
- 발문: "<보기>를 활용하여 <조건>에 따라 ㉠을 수정·보완한 것으로 가장 적절한 것은?", “다음 자료를 활용하는 방안으로 적절하지 않은 것은?(복수 자료 제시)", “개요를 수정·보완한 것으로 적절하지 않은 것은?(위계 구조 판단 포함)”
- 선택지: 자료 해석 + 글 수정·보완 복합 판단, "...하여 ~한다" 형식(계획과 초고의 관련성 제시)”
- 구조: 계획의 수정·조정 과정 반영, 복수 조건 충족 여부 판단, 복수 자료 간 종합·비교 필요, 통일성·응집성 등 텍스트 구조 차원의 판단
- 예시: "덕종어보 환수와 관련된 내용을 언급하여 '서론'을 구체화하되, 독자의 관심을 유발하는 질문 형식으로 표현한다." , "따라서 종자 개발에 경쟁력 있는 종자 기업을 정책적으로 육성하되 연구 개발에 대한 투자와 지원을 강화해야 한다. 다만 개발된 종자의 장점만을 중시하여 종자를 획일화하는 것은 위험하므로 장기적 안목에서 다양한 종자의 개발과 확보에 노력해야 한다."


### 선택지 난이도 조절 예시

**낮은 난이도:**
① 해외에 있는 우리 문화재가 국내로 반환된 최근의 사례를 추가한다.
② 맞춤법이 틀린 부분을 수정한다.
③ 문단의 통일성을 위해 벗어난 문장을 삭제한다.
④ 글의 목적은 정보 전달이다.
⑤ 예상 독자는 같은 학년 학생들이다.

**높은 난이도:**
① 2문단에서는 구체적 사례를 제시하여 독자의 이해를 돕고, 3문단에서는 전문가 인터뷰를 인용하여 주장의 신뢰성을 높이고 있다. 
② 자료 (가)의 수치 증가 추세와 자료 (나)의 전문가 견해를 종합하여, 문제의 심각성과 해결 방안을 동시에 제시한다. 
③ '서론'의 ㉠을 수정할 때, 추상적 개념을 구체적 사례로 바꾸되 독자의 경험과 연결되도록 표현하고, 통계 자료를 추가하여 객관성을 높인다. 
④ 개요의 'Ⅱ-2'를 'Ⅲ-1'로 이동하면, 원인 분석 후 바로 해결 방안을 제시하는 논리적 흐름이 형성되어 글의 응집성이 높아진다. 
⑤ ㉠은 피동 표현을 능동 표현으로 바꾸고, ㉡은 추상적 표현을 구체적 수치로 교체하며, ㉢은 접속어를 추가하여 문장 간 연결을 강화한다.

### 단계별 난이도 조절 기법

 #### 1) 계획하기 단계
 | 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) | 
|----------------|------------------| 
| 목적, 독자, 주제 중 하나만 판단 | 목적+독자+매체+형식 복합 판단 |
| "글의 목적은 무엇인가?" | "이 계획이 목적과 독자를 동시에 고려했는지 판단" | 
| 계획 요소의 직접 확인 | 계획 요소 간 정합성 판단 (목적과 매체의 적합성 등) |

 #### 2) 내용 생성하기 단계
| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) | 
|----------------|------------------| 
| 단일 자료의 내용 확인 | 복수 자료의 종합·비교 |
| 자료에 명시된 정보 그대로 활용 | 자료 해석 + 추론 필요 (예: 그래프 추세 분석) |
 | "자료에서 확인할 수 있는 내용은?" | "자료 (가)와 (나)를 종합하여 도출할 수 있는 결론은?" |

 #### 3) 내용 조직하기 단계 
| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) | 
|----------------|------------------|
 | 개요의 표면적 순서 확인 | 상위-하위 항목의 위계 관계 판단 |
 | "서론-본론-결론 구조인가?" | "하위 항목이 상위 항목을 논리적으로 뒷받침하는가?" |
 | 단일 항목의 적절성 판단 | 항목 이동 시 전체 흐름에 미치는 영향 판단 | 

#### 4) 표현하기 단계 
| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) | 
|----------------|------------------| 
| 명백한 맞춤법·문법 오류 찾기 | 문맥상 어색한 어휘·표현 판단 |
 | "틀린 맞춤법을 찾으시오" | "㉠을 문맥에 맞게 다듬은 것은?" | 
| 단일 문장 수준의 판단 | 문단 간 연결 및 호응 관계 판단 |

 #### 5) 고쳐쓰기 단계 
| 난이도 ↓ (쉽게) | 난이도 ↑ (어렵게) | 
|----------------|------------------|
 | 단일 조건 충족 | 복수 조건 동시 충족 (형식+내용) | 
| "비유법을 사용하여 고쳐 쓰시오" | "비유법을 사용하고, 구체적 수치를 포함하며, 독자의 공감을 이끌어내도록 고쳐 쓰시오" |
 | 피드백 내용의 직접 반영 | 피드백 의도 파악 + 적절한 수정 방향 판단 |
 | 어휘·문장 수준 수정 | 문단·글 전체 수준의 재구성 |


### 쓰기 문항 설계 시 유의점

- 글의 생산 과정을 중심으로 설계하되, 5단계 중 하나의 단계만 명확히 측정하도록 한다.
- 계획→초고→수정의 역동적 과정을 반영한다. 
- 고쳐쓰기는 어휘/문장/문단/글 수준을 고르게 포함한다. 
- 단순 정보 조회가 아닌 과정 간 관련성을 묻도록 설계한다.
- 난이도 '상' 이상에서는 복수 조건 충족, 복수 자료 종합, 텍스트 구조 차원의 판단을 포함한다. 
- 오답은 '조건의 부분 이행', '과잉 수정', '수정 방향 역전' 등 쓰기 과정에서 실제 발생 가능한 오류로 설계한다.

<보기>
